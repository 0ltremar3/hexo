输入输出

cin

  • 可同时输入多个变量

  • 输入一整行数组要用getline

  • cin的结束标志:
    读入空格和回车；
    while(cin >> x)时，ctrl+z退出循环。

    cout

  • 可同时输出多个变量

  • 加空格、字符用” “

  • 换行用”\n”或endl

注意事项

考试除非必要(如使用string时)，不建议用cin和cout，耗时过长，性能比较差。

STL

vector

定义

  • 头文件

  • 功能
    类似变长数组,大小随输入变化。

  • 声明

  • 类似二维数组的形式
    vector <vector<typeName> > name;//注意> >之间需要加空格，否则会是别的含意

  • vector数组
    vector <int> vi[100];//vi[0]~vi[99]每一个都是vector容器

    元素访问

    1.下标访问
    vi[index];
    2.迭代器访问
